Getting started with WRIMS2_VScript


(1) Extract the file "wrims2_vscript_v1.3_rXXXX.exe" into a directory with no spaces in the directory name

(2) Press Shift + right click to open command window in the installed folder, i.e., "wrims2_vscript_v1.3_rXXXX"

(3) Type "vscript example_positionAnalysis.py" in the command window to run the example

(4) Open ./studies/callite_PA/Run/DSS/D1641_PA_result.DSS to see results
